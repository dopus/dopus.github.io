建立本站用到的工具/服务

主要工具、服务

  • Hugo：将MarkDown文档编译成静态网页，编译时采用的主题为Learn；
  • GitHub Pages：托管编译成的静态网页；
  • git：推送更新
  • ConEmu：多标签控制台

文档、图片工具

  • Typora：写作相应的MarkDown文档；
  • PicGo-Core：从Typora中上传文档中图片
  • FastStone Capture：截图和简单的图片标注；
  • pngquant：压缩png图片；
